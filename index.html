«Ода дня одного программиста»

Знай, это не бред юмориста!
Ода дня одного программиста.
(прим. авт.)
Когда, уставший, придёшь после рабочей недели.
Но с стоном и с скрипом поднявшись с постели.
Откроешь VS себе под вечор.
И тут же получишь Unknown Error!

Что б глупостей подобных избежать.
Надо бы оду эту прочитать!
(и переходить в GitHub)

Во-первых, и это закон!
Заучи команду git clone
Затем, чтобы не съехать в «Хувейт»
Комбо из composer install и composer update
Если ты не «умелый лихач»
Посмотри, что покажет git branch
Следом, и это не просто слова:
Посмотри что покажет git branch с флагом -a
Если ты создаешь новое место,
git checkout делай по традиции test`а
Не забудь провести разведку,
На предмет именования ветки
Потом испытай удачу
Скормив название git branch`у
Не случился чтоб «Каминг-аут»:
Не забудь следом сразу checkout
А затем, чтоб пресечь неудачу
Снова глянь что отдаст по git branch`у
Просидев над задачею ночку
Добавляй все: git add . поставь точку
Проясни основные моменты:
git commit -m "[Здесь комменты]"
Чтоб коммит твой ничё не погнул
Протестируй команду git pull
Если ж код твой не полная чушь
Ждем на сервер командой git push...
Скоро кончится бешеный квест.
Репозиторий. Затем merge request.
С чистой совестью кофе налью
Лишь когда завершат code review.
А уж если и тесты пройдет,
То позвольте пожаловать в Prod.
Вот такой вот удачный денек,
Если каждый этап будет «ok».
